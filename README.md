# 📊 Business Insights 360: Retail & E-commerce Analytics for AtliQ Hardwares  

*A Power BI project unifying sales, finance, supply chain, and marketing data to improve transparency, forecasting, and profitability for a fast-growing hardware company.*

---

## ❗ Problem Statement  

**AtliQ Hardwares**, a fast-growing PC and electronics manufacturer, operates across **brick-and-mortar** and **e-commerce** channels in 27 global markets.

As the business scaled, the company faced several challenges:

- Fragmented reporting across **sales, finance, supply chain, and marketing**  
- Heavy reliance on **Excel-based static reports**  
- Limited visibility into **Latin American market performance**  
- Slower decision-making compared to analytics-driven competitors like **Dell**  

To overcome these issues, AtliQ needed a **centralized and interactive Power BI solution** to enable faster and more informed decision-making.

---

## 📝 Project Overview  

This project focuses on building a **360° Power BI analytics solution** by integrating multi-year data across business functions.

### 🎯 Objective  
Enable business leaders to:  
- Track key performance indicators (KPIs)  
- Identify performance gaps  
- Improve forecasting accuracy  
- Make data-driven decisions  

---

## 🚀 Solution  

Developed a **multi-page Power BI dashboard suite** supported by a unified data model.

### 🛠️ Tools & Technologies  
- Power BI  
- SQL  
- DAX  
- Excel

## 🔢 Data Sources  

**Primary Source:** SQL Database  
**Supplementary Source:** Excel files (benchmarks and financial targets)

### 📂 Dimensional Tables  

| Table | Description |
|------|-------------|
| `dim_customer` | 75 customers across 27 markets (Retail + E-commerce) |
| `dim_market` | 27 markets grouped into 7 sub-zones and 4 global regions |
| `dim_product` | Product hierarchy across divisions and categories |

---

### 📂 Fact Tables  

| Table | Description |
|------|-------------|
| `fact_sales_monthly` | Monthly sales data (2018–2022) |
| `fact_forecast_monthly` | Forecast data for accuracy comparison |
| Cost Tables | Freight, gross price, manufacturing cost |
| Deduction Tables | Pre-invoice and post-invoice deductions |

---

### 📊 Supplementary Data  
- Competitor market share  
- Sales and profit targets  
- Operating expenses  

---

## 📊 Dashboard Structure  

| View | Focus Area | Example KPIs |
|------|------------|--------------|
| **Finance** | Profitability and P&L analysis | Net Sales, GM%, Net Profit% vs PY |
| **Sales** | Customer and product performance | GM vs COGS, GM variance vs PY |
| **Marketing** | Operating expense efficiency | Net Profit variance vs PY |
| **Supply Chain** | Forecast performance | Forecast Accuracy, Net Error |
| **Executive Summary** | High-level business overview | Sales, Profit %, Top markets/products |
